# Architecture Decision Records

Decisions already made are captured in the design doc's [Decision Log (§5)](../design.md#5-decision-log) — that table is the baseline ADR set for v2.

New decisions (including resolving the [open questions (§20)](../design.md#20-open-questions)) get one file each here:

```
NNNN-short-title.md
```

Template:

```markdown
# NNNN — Title

**Status:** proposed | accepted | superseded by NNNN · **Date:** YYYY-MM-DD

## Context
## Decision
## Consequences
```
